# Shoot Things Until You Die

Top-down shooter. Roam around the map shooting enemies until they kill you. Maximise your score; the quicker you kill the enemies, the higher your score multiplier will get.

## To run
- `npm install`
- `npm run dev`

## Controls

`W` - Move up  
`A` - Move left  
`S` - Move down  
`D` - Move right  
`R` - Reload  
`1` - Pistol  
`2` - Model 1887  
`3` - Shotgun  
`4` - Assault rifle
